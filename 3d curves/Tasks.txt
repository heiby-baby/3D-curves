Разработайте небольшую программу на C++, которая бы реализовывала поддержку иерархии трехмерных кривых.		

		1. Поддерживайте несколько типов трехмерных геометрических кривых – окружности, эллипсы и трехмерные спирали. (Упрощенные
	определения приведены ниже). Каждая кривая должна быть способна возвращать трехмерную точку и первую производную (трехмерный
	вектор) для каждого параметра t вдоль кривой. 
		2. Заполните контейнер (например, вектор или список) объектами этих типов, созданными случайным образом, со
	случайными параметрами. 
	3. Выведите координаты точек и производные всех кривых в контейнере при t=PI/4. 
	4. Заполните второй контейнер, который будет содержать только круги из первого контейнера. Убедитесь,
		что второй контейнер разделяет (т.е. не клонирует) круги из первого контейнера, например, с помощью указателей. 
	5. Отсортируйте второй контейнер в порядке возрастания радиусов окружностей. То есть первый элемент имеет
		наименьший радиус, последний - наибольший. 
	6. Вычислите общую сумму радиусов всех кривых во втором контейнере. 

(*) Дополнительные необязательные требования: 

	7. Разделите реализацию на библиотеку кривых (.dll или .so) и исполняемый файл, который использует API этой
		библиотеки. 
	8. Реализуйте вычисление общей суммы радиусов с использованием параллельных вычислений (например, OpenMP или Intel 
		Библиотека TBB). 

Требования к реализации: 

	1. Реализация должна использовать виртуальные методы. 
	2. Не содержит ни явного освобождения памяти, ни утечек памяти. 
	3. Кривые должны быть физически правильными (например, радиусы должны быть положительными). 
	4. Контейнеры и сортировка должны быть реализованы с использованием STL (библиотека стандартных шаблонов C++). 
	5. В реализации могут использоваться конструкции C++11 или более поздней версии. 
	6. Код должен быть скомпилирован любым компилятором по вашему выбору (gcc, Visual C++ и т.д.). 

Определения кривых: - - - - 

	Все кривые определяются параметрически, т.е. точка вычисляется с использованием некоторой формулы C(t). 
	Окружность является плоской в плоскости XoY (т.е. все Z-координаты равны 0) и определяется ее радиусом. 
	Эллипс является плоским в плоскости XoY и определяется двумя его радиусами вдоль осей X и Y. 
	Спираль является пространственной и определяется ее радиусом и шагом (см. рисунок ниже). Для выполнения витка требуется 2 * PI в
	параметрическом пространстве, т.е. любая точка на спирали удовлетворяет условию C(t + 2*PI) = C(t) + {0, 0, шаг}.
